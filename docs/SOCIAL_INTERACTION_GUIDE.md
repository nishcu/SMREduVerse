# Social Interaction Guide - How to Connect & Interact

## Overview

This guide explains how the social features work in the application, similar to popular social networks like Instagram, Twitter, and Facebook. Learn how to discover users, follow them, start conversations, and interact with content.

---

## üéØ How Social Feed Works

### Social Feed Location
**Path:** `/social` (accessible from sidebar ‚Üí "Social Feed")

### Feed Types

1. **For You Feed** (Default)
   - Shows all posts from all users
   - Real-time updates
   - Sorted by newest first
   - Best for discovering new content and users

2. **Following Feed**
   - Shows posts only from users you follow
   - Personalized content
   - Updates in real-time
   - Best for staying connected with people you know

3. **Trending Feed**
   - Shows posts with most engagement (likes, comments)
   - Popular content
   - Best for discovering viral content

### How Posts Work

**Post Features:**
- **Like/Unlike**: Click the heart icon to like a post
- **Comment**: Click the comment icon to add comments
- **Share**: Share posts with others
- **Follow**: Click "Follow" button on post author to follow them
- **View Profile**: Click on author's name/avatar to visit their profile

**Post Interactions:**
- Like count shows how many people liked the post
- Comment count shows number of comments
- Click on likes to see who liked the post
- Real-time updates when new likes/comments are added

## üë• How Following/Followers System Works

### Following Someone

**Method 1: From Social Feed**
1. Go to `/social` page
2. Find a post from someone you want to follow
3. Click the **"Follow"** button next to their name
4. Button changes to **"Following"**
5. You'll now see their posts in your "Following" feed

**Method 2: From Profile Page**
1. Click on any user's name or avatar
2. Go to their profile page (`/profile/[uid]`)
3. Click the **"Follow"** button in the profile header
4. Button changes to **"Following"**

**Method 3: From Search**
1. Use the search bar in the header
2. Search for a user by name or username
3. Click on the user result
4. Click **"Follow"** on their profile

**Method 4: From Followers/Following Lists**
1. Go to any user's profile
2. Click on **"X Followers"** or **"X Following"**
3. Browse the list
4. Click **"Follow"** on any user card

### Viewing Followers & Following

**To View Someone's Followers:**
1. Go to their profile page
2. Click on the **"X Followers"** link (below their name)
3. See all users following them
4. Follow/unfollow directly from the list

**To View Who Someone is Following:**
1. Go to their profile page
2. Click on the **"X Following"** link
3. See all users they follow
4. Follow/unfollow directly from the list

### What Happens When You Follow Someone?

‚úÖ **You Get:**
- Their posts appear in your "Following" feed
- You can see their activity
- You can message them (if they allow it)
- You receive notifications about their activity

‚úÖ **They Get:**
- A notification that you followed them
- Your follower count increases
- They can see you in their followers list

## üí¨ How Chat/Messaging Works

### Starting a Chat

**Method 1: From Profile Page** (Easiest)
1. Go to any user's profile page
2. Click the **"Message"** button in the profile header
3. Chat window opens automatically
4. Start typing and send messages

**Method 2: From Social Feed**
1. Go to `/social` page
2. Find a post from someone you want to message
3. Click on their name/avatar to go to their profile
4. Click **"Message"** button
5. Start chatting

**Method 3: From Search**
1. Search for a user in the header search bar
2. Click on the user result
3. Go to their profile
4. Click **"Message"** button

**Method 4: From Chat List**
1. Go to `/chats` page
2. If you've chatted before, click on the conversation
3. If not, start a new chat from their profile

### Chat Features

**Real-Time Messaging:**
- Messages appear instantly
- See when someone is typing
- See when messages are read (double checkmark)
- Online/offline status indicators

**Chat List:**
- Shows all your conversations
- Sorted by most recent message
- Shows unread message count
- Shows last message preview
- Shows online status (green dot)

**Chat Window:**
- Send text messages
- Share images/files (if enabled)
- See message timestamps
- Scroll through message history
- See read receipts

### Chat Restrictions

**Parental Controls:**
- If parental controls restrict chat, you can only message followers
- Check your settings at `/settings`
