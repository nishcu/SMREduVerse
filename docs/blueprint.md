# **App Name**: EduVerse Architect

## Core Features:

- Unified User Registration: Single Firebase Authentication with role selection on onboarding.
- Role-Based Dashboards: Custom dashboards for students, parents, tutors, partners, and super admins with tailored content and functionalities.
- Real-Time Chat with Moderation: YouTube-like real-time chat within study rooms using Firestore and Realtime Database, with typing indicators and moderation. The moderation process will use a tool to determine whether to flag certain words.
- LMS with Course Management: Course and lesson creation/management with enrollment and progress tracking. Content (video/text).
- Knowledge Coin Economy: Wallet functionality for earning/spending knowledge coins. Integrate transactions with wallets for each user.
- AI-Powered Task Generation: Use AI to generate tasks based on selected category and difficulty level.
- Parental Controls: Comprehensive parental controls: disable spending/chat/talent hub and enable activity logs. Store settings on a Firestore database.

## Style Guidelines:

- Student Theme: Background: A very light, airy blue (hsl(207 88% 94%)).
- Student Theme: Foreground (Text): A deep, dark blue (hsl(235 63% 20%)).
- Student Theme: Primary: A strong, dark blue, used for main buttons and highlights (hsl(235 63% 30%)).
- Student Theme: Secondary: A lighter, muted blue, used for secondary elements (hsl(207 80% 88%)).
- Student Theme: Accent: A vibrant purple, used for special highlights and attention-grabbing elements (hsl(291 63% 42%)).
- Student Theme: Destructive: A standard bright red for errors and deletion actions (hsl(0 84.2% 60.2%)).
- Dark Mode: Background: A very dark, near-black blue (hsl(222.2 84% 4.9%)).
- Dark Mode: Foreground (Text): A very light gray (hsl(210 40% 98%)).
- Dark Mode: Primary: The same very light gray, used for primary buttons against the dark background (hsl(210 40% 98%)).
- Dark Mode: Secondary: A dark gray-blue, used for cards and secondary surfaces (hsl(217.2 32.6% 17.5%)).
- Dark Mode: Accent: The same dark gray-blue is used as the accent color (hsl(217.2 32.6% 17.5%)).
- Dark Mode: Destructive: A muted, dark red (hsl(0 62.8% 30.6%)).
- Super Admin Theme: Background: A very dark, almost black blue (hsl(222.2 84% 4.9%)).
- Super Admin Theme: Foreground (Text): A light gray (hsl(210 40% 98%)).
- Super Admin Theme: Primary: Light gray, similar to the main dark theme (hsl(210 40% 98%)).
- Super Admin Theme: Secondary: A slightly lighter dark blue-gray for secondary elements (hsl(215 27.9% 16.9%)).
- Super Admin Theme: Destructive: A muted, dark red (hsl(0 62.8% 30.6%)).
- Body text font: 'Inter', a grotesque-style sans-serif with a modern and neutral look.
- Headline font: 'Space Grotesk', a proportional sans-serif with a computerized and techy feel.
- Use a set of modern and minimalist icons with a consistent style across the platform.
- Design a clear and intuitive layout with consistent spacing and alignment to ensure a seamless user experience.
- Incorporate subtle animations and transitions to enhance interactivity and provide visual feedback.
